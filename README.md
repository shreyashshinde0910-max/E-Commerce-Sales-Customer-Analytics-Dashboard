# E-Commerce-Sales-Customer-Analytics-Dashboard
An end-to-end E-Commerce Sales &amp; Customer Analytics project using Python, PostgreSQL, SQL, and Power BI.

🛒 E-Commerce Sales & Customer Analytics Dashboard

📌 Project Overview

The E-Commerce Sales & Customer Analytics Dashboard is an end-to-end data analytics project that transforms raw e-commerce transaction data into actionable business insights. The project demonstrates the complete analytics workflow, including data cleaning, feature engineering, SQL-based business analysis, and interactive dashboard development.

The project was built using Python, PostgreSQL, SQL, and Power BI to showcase practical data analytics skills commonly required in the industry.

🎯 Objectives

--Analyze overall sales and profitability.
--Track monthly and quarterly sales performance.
--Identify top-performing products and categories.
--Analyze customer purchasing behavior.
--Evaluate regional sales performance.
--Study payment methods and shipping modes.
--Build an interactive dashboard for business decision-making.

📂 Dataset

The project uses three datasets:

Customers – Customer information and demographics.
Orders – Transaction details, sales, profit, discounts, shipping, and payment information.
Products – Product details, categories, and pricing.
